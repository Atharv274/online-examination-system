# Online Examination System

## Project Description
Online Examination System is a web-based application developed using Java, JSP and Servlet technology to conduct and manage online examinations.

## Technology Stack and Tools Used
- Java
- JSP
- Servlet
- HTML
- CSS
- JavaScript
- MySQL
- Apache Tomcat
- NetBeans
- GitHub

## Features and Functionalities Implemented
- Student Login
- Admin Login
- Exam Management
- Question Management
- Result Generation
- Database Connectivity

## Installation / Execution Steps
1. Open project in NetBeans
2. Configure Apache Tomcat server
3. Import MySQL database
4. Run the project
5. Access application in browser
